== Description ==

WordPress plugin that shows the user a list of tags.

== Installation ==

1. Upload the `jungtag` folder to your plugins directory (e.g. `/wp-content/plugins/`)
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Follow the instructions
